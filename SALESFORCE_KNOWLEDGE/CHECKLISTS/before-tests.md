# Before Writing Tests Checklist

- [ ] Create all required records.
- [ ] Avoid relying on org data.
- [ ] Use factories for repeated setup.
- [ ] Mock callouts.
- [ ] Avoid brittle literals when automation can normalize values.
- [ ] Test blank and populated optional fields.
- [ ] Test bulk where logic is trigger/service based.
- [ ] Watch file-heavy test row locks.
- [ ] Ensure deploy-bound classes have enough branch coverage.
