# Before Writing LWC Checklist

- [ ] Read `.js-meta.xml`.
- [ ] Confirm targets and form factors.
- [ ] Confirm direct placement or parent/child role.
- [ ] Decide LDS/UI API vs Apex.
- [ ] Plan loading, empty, error, and success states.
- [ ] Plan refresh after mutation.
- [ ] Avoid template expressions.
- [ ] Use supported navigation.
- [ ] Check desktop and mobile layout.
- [ ] Use theme-safe colors.
