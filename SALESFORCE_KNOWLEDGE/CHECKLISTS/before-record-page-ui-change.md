# Before Record Page UI Change Checklist

- [ ] Identify active page by object and form factor.
- [ ] Inspect object `View` overrides.
- [ ] Inspect FlexiPage metadata.
- [ ] Inspect Aura template regions if present.
- [ ] Inspect layout and compact layout.
- [ ] Inspect action metadata and Highlights Panel.
- [ ] Confirm standard vs custom related-list behavior.
- [ ] Confirm mobile page is not different.
- [ ] Confirm permissions and FLS.
- [ ] Avoid deploying unrelated page metadata.
