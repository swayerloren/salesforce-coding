# Glossary

## Common Salesforce Terms

- Apex: Salesforce server-side language.
- Aura: older Lightning component framework, still common for templates and wrappers.
- CMDT: Custom Metadata Type.
- ContentVersion: file version row in Salesforce Files.
- ContentDocument: logical file record in Salesforce Files.
- ContentDocumentLink: link between a file and a parent record.
- Dynamic Forms: FlexiPage field-section based field rendering.
- FlexiPage: Lightning page metadata.
- LDS: Lightning Data Service.
- LWC: Lightning Web Component.
- Quick Action: metadata-defined action on a record or object.
- RunSpecifiedTests: deploy test level that runs named Apex test classes.
- Selected-class coverage: coverage Salesforce enforces for Apex classes in a deploy package.
