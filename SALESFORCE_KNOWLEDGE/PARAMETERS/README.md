# Parameter Maps

These files explain important command parameters and options Codex may encounter when validating Salesforce work.

Parameter names can change as tools evolve. Codex should verify uncertain options against installed tool help or upstream docs before execution.

## Map Files

| File | Use |
| --- | --- |
| [CODE_ANALYZER_PARAMETERS.md](CODE_ANALYZER_PARAMETERS.md) | Salesforce Code Analyzer options. |
| [LWC_JEST_PARAMETERS.md](LWC_JEST_PARAMETERS.md) | LWC Jest options. |
| [ESLINT_LWC_PARAMETERS.md](ESLINT_LWC_PARAMETERS.md) | LWC ESLint config/profile options. |
| [PRETTIER_APEX_PARAMETERS.md](PRETTIER_APEX_PARAMETERS.md) | Apex formatter options. |
| [LOCAL_AUTOMATION_PARAMETERS.md](LOCAL_AUTOMATION_PARAMETERS.md) | This repo's automation script flags. |

## Codex Rule

Document the exact options used in History when a validation command runs.
