# Code Of Conduct

This project uses a simple standard: be respectful, practical, and focused on improving Salesforce engineering quality.

## Expected Behavior

- Discuss technical issues directly and respectfully.
- Prefer clear evidence, examples, and reproducible guidance.
- Keep contributions public-safe.
- Help maintain a useful knowledge base for humans and AI coding agents.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private customer, business, org, or credential data.
- Posting secrets, internal URLs, screenshots from private systems, or real record identifiers.
- Derailing technical discussions with unrelated disputes.

## Reporting

For security or privacy issues, follow [SECURITY.md](SECURITY.md). For conduct issues, open a GitHub issue or contact the repository owner through GitHub.
